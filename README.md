# 📊 Sales & Profit Analysis using Python

## 📌 Overview
This project analyzes sales and profit data to extract meaningful insights and visualize business performance. It helps identify top-performing products and profitable regions using simple data analytics techniques.

---

## 🎯 Objectives
- Calculate total sales and profit
- Identify top-selling products
- Analyze profit by region
- Compare sales vs profit
- Visualize data using charts

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib

---

## 📁 Dataset
The dataset includes:
- Product
- Sales
- Profit
- Region

---

## 📊 Features
- Data loading and cleaning
- Sales and profit calculation
- Product-wise analysis
- Region-wise analysis
- Visualizations:
  - Bar charts
  - Pie charts
  - Comparison charts

---

## 📈 Insights
- Top-selling product identified based on sales
- Most profitable region determined
- Sales and profit trends visualized clearly
- Higher sales products tend to generate higher profit
